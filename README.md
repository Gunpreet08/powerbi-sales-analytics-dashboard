# Sales Analytics Dashboard – Power BI

## Project Overview
This project presents an interactive Sales Analytics Dashboard developed using Power BI. The dashboard helps analyze sales performance, product trends and regional insights to support data-driven business decisions.

## Key Features
- KPI indicators for Total Sales, Profit and Orders
- Monthly Sales Trend analysis
- Regional Sales comparison
- Product performance insights
- Interactive filters for region, category and order date

## Tools Used
- Power BI
- Excel / CSV dataset
- Data visualization techniques
